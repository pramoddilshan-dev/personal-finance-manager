# Personal Finance Manager

Personal Finance Manager is a Java desktop application designed to help users manage their finances by tracking income, expenses, and categories. The application provides a clean and interactive interface built with JavaFX and stores financial data securely in a MySQL database.

## Features

- User login system
- Add and manage income transactions
- Add and manage expense transactions
- Category management (Income and Expense categories)
- View all transactions in a structured table
- Edit and delete transactions
- Financial reports with charts
- Visual indicators for income and expenses
- Clean and responsive JavaFX UI

## Technologies Used

- Java (JDK 17)
- JavaFX
- MySQL
- JDBC
- FXML (UI Layout)
- IntelliJ IDEA / VS Code

## System Modules

- Authentication system
- Income management
- Expense management
- Category management
- Transaction history
- Financial reporting dashboard

## Purpose of the Project

This project was developed as part of a learning process to practice building a complete desktop application using Java, implementing MVC architecture, database integration, and a graphical user interface.
